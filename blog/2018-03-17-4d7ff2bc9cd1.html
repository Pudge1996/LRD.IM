macOS 安装 AE 插件 Flow 无法打开的修复方法

Pudge · 2018-03-17 首次发布于 Medium

以前一直在 AE2015 里使用 Flow，最近下载了 AE2018，安装好 Flow后点击 Window-Extensions-Flow 却没有任何反应，几经波折找到了解决方法。
以前一直在 AE2015 里使用 Flow，最近下载了 AE2018，安装好 Flow后点击 Window-Extensions-Flow 却没有任何反应，几经波折找到了解决方法。

2019.01.13 更新：AE2019 必须使用新版 1.4.0b 或更新版本，否则即使设置正确也无法正常使用。

2021.06.20 更新：方法适用于 Big Sur 下的 AE2020。

解决方法

在终端依次运行以下命令：

defaults write com.adobe.CSXS.5 PlayerDebugMode 1

defaults write com.adobe.CSXS.6 PlayerDebugMode 1

defaults write com.adobe.CSXS.7 PlayerDebugMode 1

defaults write com.adobe.CSXS.8 PlayerDebugMode 1

再次打开 AE，Flow 已经可以正常使用了！

PS:记得在首选项里勾选允许脚本写入文件和访问网络！
